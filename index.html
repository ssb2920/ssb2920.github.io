Work

I did by Undergrad in Computer Science and Engineering. Here are some of my projects from my undergrad and from three semesters at Georgia Tech.

Ray Tracer

Built a ray tracer that works faster by using NumPy functions.Implemented Monte Carlo Ray Tracing with importance sampling, refraction, thin-film interference, and textures.

Procedural Generation of a Dungeon, Buildings and Creatures

Created 3D dungeons, that can be used in an RPG, Roguelike or Adventure-style game. Dungeons were procedurally generated using the Cellular Automata and Marching Squares Algorithms. The player can switch between First person and Third person mode by pressing a key as they explore the dungeon. Buildings with different shapes and sizes are generated procedurally using random seeds. Creatures with random shapes and body parts are generated and animated.

Indoor Navigation System for College Premises

To build an accurate routing system, the system is divided into three parts. A complete floor plan is converted into a 3D model instead of mapping individual points sequentially. Each floor in the plan is converted into an accurate 3D model. A graph is generated based on the 3D model using the DFS algorithm to represent the navigation path to be followed. This system finds application with architects and civil engineers.
